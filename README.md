## Requirements

- CMake Version > 3.10
- G++ Version 6.3.0 **OR** Apple clang version 16.0.0

## How to compile

```bash
cd build
cmake ..
make
```

This will generate an executable approx_dist_oracle

## How to run

From the main directory,

```bash
cd build
./approx_dist_oracle
```
